# ListApp 开发心跳检查

## 检查项（每小时）

1. **GitHub 状态检查**
   - 检查 gh-pages 分支是否最新
   - 检查是否有部署失败

2. **功能验证**
   - 访问 https://fizzzli.github.io/list-app/ 确认页面正常
   - 检查 test.html 测试套件

3. **用户反馈收集**
   - 搜索 Reddit/Twitter 上的 todo app 讨论
   - 收集新功能灵感

4. **迭代开发**
   - 根据反馈开发新功能
   - 更新测试用例
   - 部署到 gh-pages
